Kitronik Inventor's Kit for Arduino v1.1

A beginner-friendly Arduino kit for learning programming and electronics. Includes all components needed to complete 10 experiments with LEDs, motors, sensors, and capacitors.

Features:
- Complete 10 guided experiments with included components.
- Small prototype breadboard for fast circuit prototyping.

Contents: 7-segment display, servo, potentiometer, push switches, motor, LEDs, resistors, jumper wires, capacitor, piezo buzzer, phototransistor, and other essential components.
